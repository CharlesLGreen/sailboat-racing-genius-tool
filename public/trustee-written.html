4. How to Keep the Sites Running (No Code Experience Required)

The sites are designed to run automatically without any intervention. In most cases, you will not need to do anything.

4.1 Normal Operations (Nothing to Do)

  • Snipeovation.com is hosted on Railway and restarts automatically if it crashes.
  • Crew College is hosted on Vercel and runs automatically.
  • User data is preserved on Railway's persistent storage.
  • Domain renewals need to be handled annually (check with the domain registrar).

4.2 If a Site Goes Down

Step 1: Check Railway (for Snipeovation.com):

  • Go to https://railway.com and log in
  • Click on the "snipe-app" project
  • Look for a red "Crashed" status
  • Click "Redeploy" or "Restart" button

Step 2: Check Vercel (for Crew College):

  • Go to https://vercel.com and log in
  • Find the Crew College project
  • Click "Redeploy" on the latest deployment

4.3 How to Make Updates Using Claude AI (Recommended Method)
